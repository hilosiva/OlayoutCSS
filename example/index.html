Ore no Layout CSS Framework

Wellcome

Olayaout CSSはWebページのレイアウトが効率コーディングできる俺流のレイアウト用CSSフレームワークです。
正直「Olayout」というネーミングは変えたいと思ってますよ。そりゃ。
このOlayaout CSSに主に以下の機能があります。

Reset

ブラウザのデフォルトスタイルで設定されている要素のリセットを行います。フォントサイズなどはリセットされません。

Container & Spacer

ボックスのセンタリングや、各ブロック間の上下の余白、ブロック内の余白をデザインに合わせて自由に設定することができます。

Grid System

各ブロック間の上下の余白や、ブロック内の余白を予めデザインに合わせて自由に設定することができます。

Install

「olayout.min.css」を読み込む。

Settings

カスタムプロパティで余白量や配色、書体を設定を上書きできる

Reset

ブラウザのデフォルトスタイルを調整するReset用のCSSが入っています。

  • - 全ての要素は box-sizing: border-box をデフォルトで継承する
  • - 不必要な margin 、 padding、 border を排除
  • - 文字サイズのリセットは行わない
  • - font-feature-settings による文字詰め
  • - 不必要な margin 、 padding、 border を排除
  • - line-heigh は 1.5 　をデフォルトにする
  • - html 要素に scroll-behavior: smooth を指定
  • - アクセシビリティに考慮

Container & Spacer

test

ボックスのセンタリングは .ol-container を ブロック間の余白には .ol-spacer を使う。

Top and bottom margins

上下の余白には、data-py属性を使い「ウインドウサイズ名：スペース名」で指定する。（半角スペース区切りで複数指定可能）

上だけ指定する場合は「ウインドウサイズ名：スペース名Top」とし、下だけ指定する場合は「ウインドウサイズ名：スペース名Bottom」とします。

Left and right margins

左右の余白には、data-x属性を使い「ウインドウサイズ名：スペース名」で指定する。（半角スペース区切りで複数指定可能）

左だけ指定する場合は「ウインドウサイズ名：スペース名Left」とし、右だけ指定する場合は「ウインドウサイズ名：スペース名Right」とします。
